PinqPonq

Back Website

PinqPonq is a webshop that sells bags of any shape, you have wide selection of colors and custom options. The goal of this design, is that the user can make their own custom bag using the website.

Year

2021

Client

Hogeschool van Amsterdam

Assignment

Our assignment was to create a design where the user can create their very own bag of choice. Using the case provided, we had a list of requirements in order to pass. All while keeping all the UX principles in mind.
