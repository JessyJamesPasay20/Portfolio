Mappa

Back Website

This is a redesign of an existing website from Mappa.co.jp. After doing research in regards to inclusivity, I had concluded that several changes had to be done to the website to assist users with different disabilities to have a better user experience. Further was the Html code of the website, not following the Semantic guidelines from W3C. So I corrected that aswell.

The website is available on all devices ( responsive ), though not all buttons and interactions lead to pages as I've only designed two pages. Further the website changes based on if the user is using Light or Dark mode.

Year

2022

Client

Hogeschool van Amsterdam

Assignment

This was my final assignment for FED ( Front end design ). The assignment was to find a website you like, and research whether it's inclusive for people with disabilities. Further you had to correct the HTML / CSS while following the W3C guidelines.
